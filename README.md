# Personal Task Tracker

A simple and user-friendly desktop application built using **Python** and **Tkinter** to help users manage their daily tasks efficiently. The application allows users to add, complete, delete, and save tasks locally using a JSON file. It also tracks the completion progress in real time.

---

## Features

- ➕ Add new tasks
- ✔ Mark tasks as completed
- 🗑 Delete selected tasks
- 💾 Automatically save tasks to a JSON file
- 📂 Automatically load saved tasks when the application starts
- 📊 Real-time progress tracking
- 📅 Display the current date
- 🖥 Simple and clean Tkinter GUI

## Getting Started

### Prerequisites

- Python 3.x installed

### Clone the Repository

```bash
git clone https://github.com/<your-username>/Personal-Task-Tracker.git
```

### Navigate to the Project Folder

```bash
cd Personal-Task-Tracker
```

### Run the Application

```bash
python main.py
```

---

## How to Use

1. Enter a task in the input box.
2. Click **Add Task** to add it to the list.
3. Select a task and click **Complete Task** to mark it as completed.
4. Select a task and click **Delete Task** to remove it.
5. The application automatically saves all changes.
6. When you reopen the application, your tasks are loaded automatically.
